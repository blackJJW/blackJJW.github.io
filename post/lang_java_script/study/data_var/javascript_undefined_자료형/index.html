[JavaScript] undefined 자료형

May 08, 2022
JavaScript

상수와 변수로 선언하지 않은 식별자

  • 상수와 변수로 선언하지 않은 식별자의 자료형을 확인하면 undefined가 나옴

    • 선언하지 않고 사용하여 undefined 자료형

값이 없는 변수

  • 변수를 선언하면서 값을 지정하지 않는 경우, 해당 식별자는 undefined 자료형이 됨

    • 상수는 선언할 때 값을 반드시 지정해야 하므로 값이 없는 상수는 존재하지 않음
